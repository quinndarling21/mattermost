# Product overview

Mattermost in this repository is a self-hosted collaboration app with channel messaging, teams, direct messages, file sharing, search, notifications, plugins, and administration through System Console.

The web app is the primary client. Native desktop and mobile clients are supported by configurable download links in System Console. Admins can hide mobile download prompts by leaving platform links blank.

## Primary product areas

- Teams group people and channels under a shared workspace.
- Channels hold public, private, direct, and group conversations.
- Threads, message priority, scheduled posts, drafts, and persistent notifications shape daily messaging behavior.
- Files use local storage by default, with optional S3 or Azure Blob storage.
- Search uses database search by default and can use Elasticsearch when licensed and configured.
- Plugins add larger product areas such as Calls, Playbooks, Boards, Apps, AI, and NPS.

## Admin surface

System Console is the main admin UI. Its sections cover reporting, user management, environment settings, site configuration, authentication, plugins, integrations, compliance, billing, and experimental features.
