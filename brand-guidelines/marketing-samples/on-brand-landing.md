# Secure collaboration for technical teams

Mattermost is an open-source collaboration platform you can host yourself. It brings messaging, workflows, and integrations into one place, so technical teams can work on sensitive projects without sending their data to a third party.

## Built for high-stakes work

- **Channels** keep real-time and threaded conversation organized by team and topic.
- **Playbooks** turn repeatable work, like incident response, into checklists that run inside your channels.
- **Integrations** connect the tools you already use through a plugin framework and webhooks.

## You control the deployment

Run Mattermost self-hosted on infrastructure you own, including air-gapped environments, or use Mattermost Cloud if you'd rather we operate it. Either way, you decide where your data lives and who can access it, with granular roles managed in the System Console.

## Open by default

Mattermost is developed in the open and built on Go and React. You can read the source, file issues, and extend the platform with plugins.
